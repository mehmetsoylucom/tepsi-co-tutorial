# Stylelint

```
yarn add stylelint --save-dev
yarn add stylelint-config-standard
```

Sisteme entegre etmek icin `.stylelintrc` dosyasi asagidaki icerikle eklenir.

```
{
  "extends": "stylelint-config-standard",
}
```

### ONEMLI NOT
Videoda anlatilan `stylelint` install dogru calismadi. Bu sekilde de vscode icinde aktif olmadi.
Simdilik `vscode plugin` olarak yukeleyip geciyorum.
